# TECH-PARK_UQ - Estructura de Datos

Proyecto para la asignatura de Estructura de Datos en la Universidad del Quindío.

## Requisitos
* **Java 21** o superior.
* **Maven** (usado como gestor de dependencias).

## Cómo ejecutar localmente
1. Clona el repositorio.
2. Abre la carpeta raíz en tu IDE preferido (IntelliJ, VS Code, Eclipse).
3. El IDE detectará el archivo `pom.xml` y descargará las librerías automáticamente.
4. Ejecuta la clase `co.edu.uniquindio.techpark.App`.

## Estructura del Proyecto
* `src/main/java`: Contiene el código fuente organizado en paquetes.
* `src/test/java`: Contiene las pruebas unitarias.
* `src/main/resources`: Recursos del proyecto (configuración, archivos txt, etc.).
